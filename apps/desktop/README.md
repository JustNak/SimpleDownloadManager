# Desktop App Scaffold

This folder is reserved for the Tauri desktop app.

Current state:

- `src/` is left open for your Svelte or other frontend UI work.
- `src-tauri/` contains backend module placeholders and protocol notes.

When you hand UI files back, include:

- file paths
- screen props and emitted events
- enum names that must match backend values
- placeholder data shapes that need backend parity
